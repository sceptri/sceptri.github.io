Explanation of L2 norm of an error

Created At：2022-09-22 19:58 Updated At：2022-09-23 17:25

When solving for the sparsest possible set of DEs, it is likely our found model will not describe the data exactly - there will be an error

Therefore we can measure the error and give the user it's l2l^2l2-norm

  • The error is a vector of errors at each time-step
  • More information regarding the l2l^2l2-norm is here

When working with 2 or more dimensional data, the l2l^2l2-norm returned will be vector of l2l^2l2 norms in each coordinate
